# Café Ordering Management System

## Description

A web-based system for managing café orders with admin and cashier roles.

## Features

* Admin: Manage meals, categories, users
* Cashier: View menu, add to cart, place orders
* Image integration using Unsplash API

## Technologies Used

* PHP
* MySQL
* HTML, CSS, JavaScript
* API (Unsplash)

## How to Run

1. Start XAMPP (Apache & MySQL)
2. Import database into phpMyAdmin
3. Place project in htdocs
4. Open: http://localhost/CafeOrderingManagementSystem
